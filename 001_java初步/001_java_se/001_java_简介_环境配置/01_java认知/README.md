* Java的就业形势
    1. 领域非常广
        * Java技术具有卓越的通用性，高效性，平台移植安全性，广泛应用于pc，数据中心，游戏控制台，科学超级计算机，移动电话和互联网，所以说Java的应用领域非常广了，同时拥有全球最大的开发者社群
    2. 需求量大
        * Java是全球最流行的开发语言我国对于Java软件工程师的需求达到全部需求的60%-70%，
    3. 就业方向非常广
        * 学完Java可以应聘java/h5/android/等职位
