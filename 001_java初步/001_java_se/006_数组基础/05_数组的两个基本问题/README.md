# 两个问题
1. 思考java中的参数传递问题以及图解
    * 基本数据类型的值传递，不改变原值，因为调用后就会弹栈，局部变量随之消失。
    * 引用数据类型的值传递，改变原值，因为即使方法弹栈，但是堆内存对象还在，可以通过地址继续访问。
        *因为引用数据类型存放的是一个内存地址   
    * java中到底是传递值还是传地址?
        1. 既是传值，也是传递地址，基本类型传递值，引用数据类型传递地址
        2. java中只有传值，因为地址值也是值，
